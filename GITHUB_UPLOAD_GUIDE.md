# GitHub Upload Guide

## Repository Settings

- Repository name: `sijasta-transaction-management`
- Visibility: Public
- Description: `Role-based transaction, inventory, stock, and reporting system built with Laravel 10 and MySQL.`
- Do not initialize the repository with another README because this project already contains one.

## Recommended Topics

`laravel` `php` `mysql` `transaction-management` `inventory-management` `role-based-access-control` `bootstrap` `web-application`

## First Upload with Git

```bash
cd sijasta-transaction-management
git init
git branch -M main
git add .
git commit -m "feat: publish sanitized SIJASTA portfolio project"
git remote add origin https://github.com/sulthanrhnn/sijasta-transaction-management.git
git push -u origin main
```

## Recommended Follow-up Commits

```text
docs: add application screenshots
test: expand transaction workflow coverage

```

After upload, pin this repository on the GitHub profile and replace the SIJASTA project text in the profile README with a direct repository link.
